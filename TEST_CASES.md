# Linux二级文件系统测试用例

## 测试环境
- 操作系统：Ubuntu 20.04+ / WSL2
- 编译器：GCC 9+
- 依赖库：ncurses5-dev

## 测试步骤

### 步骤1：编译项目
```bash
cd osfs
make clean
make
```

### 步骤2：运行测试脚本
```bash
./test_all.sh
```

### 步骤3：手动测试（推荐）
```bash
./filesys
# 选择模式1（命令行）进行交互式测试
```

## 三、虚拟内存映射测试

### 3.1 文件映射
| 用例ID | TC-020 |
|--------|--------|
| 测试项 | 将文件映射到内存 |
| 前置条件 | 文件已打开（fd=0） |
| 测试步骤 | 输入 `mmap 0 1024` |
| 期望结果 | 显示"映射成功: addr=0x..." |

### 3.2 查看映射列表
| 用例ID | TC-021 |
|--------|--------|
| 测试项 | 查看所有mmap区域 |
| 前置条件 | 已创建内存映射 |
| 测试步骤 | 输入 `mlist` |
| 期望结果 | 显示映射区域列表，包含地址、inode、长度、模式 |

### 3.3 解除映射（写回）
| 用例ID | TC-022 |
|--------|--------|
| 测试项 | 解除映射并写回修改 |
| 前置条件 | 文件以读写模式映射 |
| 测试步骤 | 输入 `munmap 0x...`（地址需替换为实际映射地址） |
| 期望结果 | 显示"已将 x 字节写回文件"和"munmap成功" |

### 3.4 同步映射内容
| 用例ID | TC-023 |
|--------|--------|
| 测试项 | 将映射内容同步写回文件 |
| 前置条件 | 文件已映射 |
| 测试步骤 | 输入 `msync 0x...`（地址需替换为实际映射地址） |
| 期望结果 | 显示"msync成功: 已同步 x 字节" |
